# Real Estate Management System - Week 03

## Description

This project implements a Real Estate Management System using modern ES2023 JavaScript features, including:

- Abstract classes
- Inheritance
- Private fields (#)
- Static blocks
- Encapsulation
- CRUD operations
- Search and filtering
- DOM integration

---

## Class Architecture

BaseProperty (abstract)
├── House
├── Apartment
└── CommercialProperty

Person
├── Agent
└── Client

RealEstateSystem
├── Transaction


---

## Features Implemented

- Abstract base class with private fields
- Three derived property types
- User system with roles (Agent and Client)
- Transactions system
- Static configuration block
- CRUD operations
- Search by name
- Filter by type
- Dynamic DOM rendering

---

## Technologies

- HTML5
- CSS3
- JavaScript ES2023
